Styrolimb V2 robotic arm by DIYGary (Aka BeautiCode)

This is just a simple robotic arm which is remotely controlled by my own DirectTV remote, and also has a feature allowing it to be programmed to recursively visit arm positions (loop)

Buttons:
Up
Down
Left
Right
Select (Move claw in/out)
1 (Enter programming mode / Log arm position / Exit looping mode)
2 (Delete logged arm position)
3 (Enter looping mode)
